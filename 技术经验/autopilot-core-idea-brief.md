# Autopilot 核心思路（简要版）

## 一句话

**确定性轨道 + LLM 引擎 + 持久记忆 + 独立评判 = 自驱飞轮。**

## 核心架构

```
  ┌──────────────────────────────────────────┐
  │                                          │
  │   State（我在哪？）                       │
  │       ↓                                  │
  │   Classify（下一步干什么？）              │
  │       ↓                                  │
  │   Dispatch（干活）                        │
  │       ↓                                  │
  │   Audit（干得怎么样？）← 独立评判器       │
  │       ↓                                  │
  │   Summary（记录 + 学习）                  │
  │       ↓                                  │
  │   State（更新后）──→ 下一轮               │
  │                                          │
  └──────────────────────────────────────────┘
```

飞轮转一圈 = 一个 **tick** = 一次完整的 Plan → Work → Review → Release。心跳自动触发，不需要人按按钮。

## 四根支柱

**1. 确定性轨道** — 不让 LLM 判断规则，规则写死在 CLAUDE.md 里（SHALL/MUST 关键字），LLM 每次会话必读必遵守。调度预算：per-tick 16次、per-day 500次、per-task 10次。迭代/修订独立计数，上限翻倍（10次/6次）。超预算直接跳到 Summary。

**2. 持久记忆** — 三层：state/（机器状态）→ diary/（日记）→ MEMORY.md（蒸馏智慧，每5 tick 提炼一次）。第100次 tick 比第1次更聪明，靠的是记忆不是模型。

**3. 独立评判** — 生成器和评判器不共享上下文。评判器默认怀疑态度，≥80分过，<60隔离。二次复核用独立 subagent 全新上下文再验一次——换上下文不换指令。

**4. 心跳机制** — 用户不在：Cron 每30分钟触发；用户在聊：主 REPL 结尾检查条件后后台派 subagent。人走不停。

## 核心文件

| 文件 | 角色 |
|------|------|
| `CLAUDE.md` | 宪法，LLM 必读的硬约束 |
| `kanban/TODO→INPROGRESS→DONE` | 任务板 + DAG 依赖 |
| `state/tick-counter.json` | 时钟 |
| `state/dispatch-budget.json` | 调度钱包（确定性计数，非估算） |
| `specs/` + `changes/` | Spec 先行，Delta 变更管理 |
| `.claude/memory/MEMORY.md` | 长期记忆 |

## 反认知投降

> 循环是忠实的乘号——理解被放大，偷懒也被放大。执行可外包，拿主意不行。

光循环不够。循环 + 轨道 + 评判 + 记忆 = 自驱。缺一则飞轮倒转。

## 一图总结

```
              ┌───────────────┐
              │  确定性轨道    │  ← CLAUDE.md 写死规则
              │ (调度/门禁)   │     LLM 每次必读必遵守
              └───────┬───────┘
                      │ 约束
                      ↓
         ┌─────────────────────────┐
         │    飞轮（Tick 循环）      │  ← LLM 做创意工作
         │  State→Classify→Dispatch │     心跳驱动不停转
         │  →Audit→Summary→State   │
         └────┬──────────────┬─────┘
              │              │
         ┌────↓─────┐  ┌────↓──────┐
         │ 持久记忆  │  │ 独立评判   │
         │(越干越懂) │  │(换人说不)  │
         │3层:state  │  │2层:evaluate│
         │  →diary   │  │  →verify  │
         │  →MEMORY  │  │           │
         └──────────┘  └───────────┘
```

**飞轮 + 轨道 + 记忆 + 评判 = 自驱系统。**
